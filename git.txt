SETUP
Instaleer: https://git-scm.com/downloads !!!!!
Start opdrachtprompt (CMD) en voer de twee hieronder commands uit
git config --global user.name "Jouw naam hier"
git config --global user.email "jouw github email"

ALLEEN NAAR DE MASTER PUSHEN ALS HET ECHT WERKT!!!!

git clone https://github.com/PiekJ/I-Project-EenmaalAndermaal.git
 - hoeft maar eenmalig, letop dat deze een map maakt genaamt I-Project-EenmaalAndermaal in de huidige console map (om deze te veranderen typ: cd /map/naar/waar/je/heen/wilt en om van schrijf te veranderen typ: D:, C: etc)

gitk
 - grafische weergave van je wijzigingen

git status
 - geeft je de status van (on)gecommitted veranderingen

git add .
 - voegt alle gewijzigde bestanden toe aan een commit

git commit -m "Hier je commit bericht"
 - maakt een commit aan om die veranderingen naar de "master" te sturen.

git push origin master
 - stuurt de commit naar de github service

git pull origin master
 - haalt alle bewerkingen op van de github service en probeert deze samen te voegen met jouw bewerkingen
